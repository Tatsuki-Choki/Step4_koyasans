# WEBアプリ要件定義書：チームコンパス

## 1. プロジェクト概要

### 1.1 プロジェクト名
チームコンパス（仮称）

### 1.2 プロジェクトの目的
「チャレンジする心」「諦めない力」「やり遂げる力」をデータで見える化し、社会で通用する力を育てる教育DXを実現する

### 1.3 ビジョン
下妻第一高校の生徒が、このアプリを通じて非認知能力を向上させ、社会に出たときに「社会的なスキルが高い」「戦闘力が高い」と評価される人材になれるよう支援する

### 1.4 ターゲットユーザー
- **プライマリユーザー**: 下妻第一高校の生徒（高校1年生〜3年生）
- **セカンダリユーザー**: 下妻第一高校の教員

## 2. 機能要件

### 2.1 コア機能

#### 2.1.1 生徒側機能

**ジャーナリング機能**
- 生徒が日々の行動や発言、思考内容を記録できる
- ゲーム化要素を取り入れ、楽しみながら継続できるUI
- テキスト入力による記録
- チャレンジした行動、他者を助けた記録などを記録可能

**成長の可視化機能**
- AIが分析した非認知能力の推移をグラフで表示
- 「チャレンジする心」「諦めない力」「やり遂げる力」の各指標を可視化
- 自分を高める力（意欲、向上心、自信、楽観性）
- 自分と向き合う力（自制心、忍耐、レジリエンス）
- 他者と繋がる力（コミュニケーション能力）
- アントレプレナーシップに関連する能力の可視化

**振り返り機能**
- 過去の記録を振り返れる
- AIが分析した内容を確認できる
- 自分の成長を実感できる

**目標設定機能**
- 生徒が目標を設定できる
- 先生と目標を共有できる仕組み（ジョイントビジネスプラン的な思想）

#### 2.1.2 教員側機能

**ダッシュボード機能**
- 全生徒の状況を一覧で確認できる
- 生徒一人ひとりの努力や変化を可視化
- 非認知能力の推移を確認できる

**個別サポート機能**
- 生徒一人ひとりの詳細情報を確認できる
- 個別に声がけできるように支援情報を表示
- 生徒の目標を共有し、一緒に達成を目指す

**アラート機能**
- 気になる生徒の変化を通知
- サポートが必要な生徒を特定

### 2.2 AI分析機能

#### 2.2.1 分析対象
- 生徒の行動データ
- 発言内容
- ジャーナリングの内容
- 思考パターン

#### 2.2.2 分析内容
- 非認知能力の評価
- 社会で通用する力の評価
- アントレプレナーシップに関連する能力の評価
- 成長の傾向分析

#### 2.2.3 分析の精度
- 教育学・心理学の専門家のサポートを受けた分析ロジック
- 下妻高校の先生たちとの協力による実践的改善

### 2.3 データ管理機能

#### 2.3.1 データ保存
- 生徒の記録データを安全に保存
- 個人情報保護に配慮した設計

#### 2.3.2 データ活用
- 卒業後の進学・就職活動で活用できるレポート生成
- 面接などで使える客観的なデータの提供

## 3. 非機能要件

### 3.1 パフォーマンス
- ページの読み込み時間: 3秒以内
- レスポンス時間: 1秒以内

### 3.2 セキュリティ
- 個人情報の適切な管理
- 学校環境での利用に適したセキュリティレベル
- プライバシー保護

### 3.3 ユーザビリティ
- 高校生が直感的に操作できるUI
- ゲーム化要素による継続性の向上
- 教員が効率的に管理できるUI

### 3.4 アクセシビリティ
- モバイルデバイス対応
- タブレット対応
- PC対応

## 4. UI/UX要件

### 4.1 デザインコンセプト
- muute（https://muute.jp/）を参考にした洗練されたUI
- 静かで優しいデジタル空間の提供
- ゲーム化要素を取り入れた楽しい体験

### 4.2 生徒側UI要件

**ジャーナリング画面**
- シンプルで書きやすい入力画面
- ゲーム化要素（バッジ、レベル、ポイントなど）
- 継続を促すデザイン

**ダッシュボード画面**
- 成長を視覚的に確認できるグラフ
- わかりやすい指標表示
- モチベーションを高めるデザイン

**振り返り画面**
- 過去の記録を時系列で確認できる
- AI分析結果をわかりやすく表示

### 4.3 教員側UI要件

**ダッシュボード画面**
- 全生徒の状況を一覧で確認できる
- フィルタリング機能
- 検索機能

**個別生徒画面**
- 生徒の詳細情報を確認できる
- 成長の推移をグラフで表示
- サポート情報の表示

### 4.4 UIライブラリ
- モダンなUIライブラリの使用を検討
- レスポンシブデザイン対応

## 6. 競合分析と差別化ポイント

### 6.1 競合サービス

**エデュパス**
- 非認知能力を伸ばすことに重きを置く
- コーチング機能が充実
- 可視化よりも伸ばすことに重点

**AIグロー**
- 可視化に重きを置く
- 点数化・コンピテンシー評価に重点
- 非認知能力をどうにかして数値化

**muute**
- 思考と感情の可視化
- UIが優れている
- muuteスクール版はB2Bで先生向けダッシュボードあり

### 6.2 差別化ポイント

1. **アントレプレナーシップに焦点**
   - 社会で生き抜く力、ビジネスで戦える力を重視
   - 他のサービスは非認知能力全般に焦点

2. **ジョイントビジネスプラン的な思想**
   - 生徒と先生が目標を共有し、一緒に達成を目指す
   - 共同コミットメントの仕組み

3. **社会で通用する人材の育成**
   - 最終的に「下妻第一高校生は社会で活躍する人材を排出する学校」という評価を得る
   - 進学・就職活動で活用できるレポート生成

## 7. リスクと課題

### 7.1 プライバシーに関する課題
- 生徒が本音を書かない可能性
- 保護者や教員の個人情報への懸念
- 学校環境での利用に対する抵抗感

**対策**
- ゲーム化により楽しさを提供し、抵抗感を軽減
- 個人情報保護への配慮を明確に示す
- 生徒と教員の信頼関係の構築

### 7.2 ユーザー継続性の課題
- 高校生が継続的に利用するかどうか
- 受験勉強優先で利用されない可能性

**対策**
- ゲーム化による継続性の向上
- 無意味と思わせないデザイン
- 修学旅行や文化祭のように楽しめる要素の追加

### 7.3 AI分析の精度
- AI分析のアウトプットの質にばらつきがある可能性

**対策**
- 教育学・心理学の専門家のサポート
- 実践的な改善を重ねる
